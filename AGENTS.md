# Lily Design System - Angular Headless

Angular implementation of the Lily Design System headless component library. Currently a placeholder — implementation work has not yet begun.

When implementation starts, mirror the conventions of the existing headless libraries:

- Angular 18+ with standalone components
- TypeScript
- Each component as a class with template + tests
- Headless: no CSS, no styles
- Component file structure: `{ComponentPascalCase}.component.ts` + `.spec.ts` + `.md`
