# DropdownMenu

a menu that opens below a trigger button

This is the Angular headless implementation. See `components/dropdown-menu/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-dropdown-menu></lily-dropdown-menu>
```

## Files

- `DropdownMenu.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `DropdownMenu.spec.ts` — vitest + TestBed render test
