# BannerBox

a banner box that is inside a banner component, using flexbox horizontal

This is the Angular headless implementation. See `components/banner-box/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-banner-box></lily-banner-box>
```

## Files

- `BannerBox.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `BannerBox.spec.ts` — vitest + TestBed render test
