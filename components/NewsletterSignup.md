# NewsletterSignup

an email subscription form composition with idle, submitting, success, and error states

This is the Angular headless implementation. See `components/newsletter-signup/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-newsletter-signup></lily-newsletter-signup>
```

## Files

- `NewsletterSignup.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `NewsletterSignup.spec.ts` — vitest + TestBed render test
