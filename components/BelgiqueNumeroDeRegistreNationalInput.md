# BelgiqueNumeroDeRegistreNationalInput

an input for entering Belgium's Numéro de Registre National / Rijksregisternummer (NRN)

This is the Angular headless implementation. See `components/belgique-numero-de-registre-national-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-belgique-numero-de-registre-national-input></lily-belgique-numero-de-registre-national-input>
```

## Files

- `BelgiqueNumeroDeRegistreNationalInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `BelgiqueNumeroDeRegistreNationalInput.spec.ts` — vitest + TestBed render test
