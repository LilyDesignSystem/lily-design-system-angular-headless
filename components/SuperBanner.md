# SuperBanner

a super-banner communicates a high-priority state that affects an entire app, experience, process, or system

This is the Angular headless implementation. See `components/super-banner/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-super-banner></lily-super-banner>
```

## Files

- `SuperBanner.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SuperBanner.spec.ts` — vitest + TestBed render test
