# Hero

a large box or image with a title and description

This is the Angular headless implementation. See `components/hero/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-hero></lily-hero>
```

## Files

- `Hero.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Hero.spec.ts` — vitest + TestBed render test
