# GovernmentIdentifier

an identifier section with a parent agency logo, agency name, and required government links

This is the Angular headless implementation. See `components/government-identifier/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-government-identifier></lily-government-identifier>
```

## Files

- `GovernmentIdentifier.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `GovernmentIdentifier.spec.ts` — vitest + TestBed render test
