# SlovenskoRodneCisloView

a read-only display of Slovakia's Rodné číslo (RČ)

This is the Angular headless implementation. See `components/slovensko-rodne-cislo-view/index.md`
in the canonical repo root for the cross-framework documentation.


- `SlovenskoRodneCisloView.spec.ts` — vitest + TestBed render test
