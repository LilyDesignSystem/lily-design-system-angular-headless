# ArticleLayout

a top-level article wrapper that sets CSS custom properties for content column widths

This is the Angular headless implementation. See `components/article-layout/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-article-layout></lily-article-layout>
```

## Files

- `ArticleLayout.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ArticleLayout.spec.ts` — vitest + TestBed render test
