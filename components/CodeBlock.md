# CodeBlock

a block of formatted code with optional line numbers and line highlighting

This is the Angular headless implementation. See `components/code-block/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-code-block></lily-code-block>
```

## Files

- `CodeBlock.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `CodeBlock.spec.ts` — vitest + TestBed render test
