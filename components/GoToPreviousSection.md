# GoToPreviousSection

a link that takes users to the previous section of the current page

This is the Angular headless implementation. See `components/go-to-previous-section/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-go-to-previous-section></lily-go-to-previous-section>
```

## Files

- `GoToPreviousSection.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `GoToPreviousSection.spec.ts` — vitest + TestBed render test
