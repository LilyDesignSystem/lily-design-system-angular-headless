# RadioGroup

a group of radio buttons for selecting one option

This is the Angular headless implementation. See `components/radio-group/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-radio-group></lily-radio-group>
```

## Files

- `RadioGroup.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `RadioGroup.spec.ts` — vitest + TestBed render test
