# MockupTabletPortrait

a box area that looks like a tablet computer in portrait mode

This is the Angular headless implementation. See `components/mockup-tablet-portrait/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-mockup-tablet-portrait></lily-mockup-tablet-portrait>
```

## Files

- `MockupTabletPortrait.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `MockupTabletPortrait.spec.ts` — vitest + TestBed render test
