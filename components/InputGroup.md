# InputGroup

a group wrapping an input with prefix and suffix addons

This is the Angular headless implementation. See `components/input-group/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-input-group></lily-input-group>
```

## Files

- `InputGroup.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `InputGroup.spec.ts` — vitest + TestBed render test
