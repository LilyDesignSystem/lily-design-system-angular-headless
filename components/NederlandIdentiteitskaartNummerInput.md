# NederlandIdentiteitskaartNummerInput

an input for entering Netherlands's Identiteitskaart Nummer

This is the Angular headless implementation. See `components/nederland-identiteitskaart-nummer-input/index.md`
in the canonical repo root for the cross-framework documentation.
