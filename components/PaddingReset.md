# PaddingReset

a container that resets padding inside fluid-width containers

This is the Angular headless implementation. See `components/padding-reset/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-padding-reset></lily-padding-reset>
```

## Files

- `PaddingReset.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `PaddingReset.spec.ts` — vitest + TestBed render test
