# LiechtensteinNationalIdentityCardNumberView

a read-only display of Liechtenstein's Liechtenstein National Identity Card Number

This is the Angular headless implementation. See `components/liechtenstein-national-identity-card-number-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-liechtenstein-national-identity-card-number-view></lily-liechtenstein-national-identity-card-number-view>
```

- `LiechtensteinNationalIdentityCardNumberView.spec.ts` — vitest + TestBed render test
