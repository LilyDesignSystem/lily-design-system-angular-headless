# ContainerWithFixedWidth

a centered content wrapper with a fixed max-width breakpoint

This is the Angular headless implementation. See `components/container-with-fixed-width/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-container-with-fixed-width></lily-container-with-fixed-width>
```

## Files

- `ContainerWithFixedWidth.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ContainerWithFixedWidth.spec.ts` — vitest + TestBed render test
