# GrailLayout

a responsive web design structure with header, left aside, center main, right aside, footer

This is the Angular headless implementation. See `components/grail-layout/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-grail-layout></lily-grail-layout>
```

## Files

- `GrailLayout.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `GrailLayout.spec.ts` — vitest + TestBed render test
