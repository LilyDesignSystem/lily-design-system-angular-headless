# ToggleGroup

a group of toggle buttons for selecting options

This is the Angular headless implementation. See `components/toggle-group/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-toggle-group></lily-toggle-group>
```

## Files

- `ToggleGroup.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ToggleGroup.spec.ts` — vitest + TestBed render test
