# MutuallyExclusive

a container ensuring only one option can be selected from a group

This is the Angular headless implementation. See `components/mutually-exclusive/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-mutually-exclusive></lily-mutually-exclusive>
```

## Files

- `MutuallyExclusive.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `MutuallyExclusive.spec.ts` — vitest + TestBed render test
