# Meter

a gauge displaying a scalar value within a known range

This is the Angular headless implementation. See `components/meter/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-meter></lily-meter>
```

## Files

- `Meter.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Meter.spec.ts` — vitest + TestBed render test
