# Emoji

a single emoji character with accessible name

This is the Angular headless implementation. See `components/emoji/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-emoji></lily-emoji>
```

## Files

- `Emoji.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Emoji.spec.ts` — vitest + TestBed render test
