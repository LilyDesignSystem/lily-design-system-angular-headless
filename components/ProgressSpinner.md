# ProgressSpinner

an indeterminate spinning progress indicator

This is the Angular headless implementation. See `components/progress-spinner/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-progress-spinner></lily-progress-spinner>
```

## Files

- `ProgressSpinner.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ProgressSpinner.spec.ts` — vitest + TestBed render test
