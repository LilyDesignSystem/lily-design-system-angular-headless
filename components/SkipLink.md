# SkipLink

a hidden link for keyboard users to skip to main content

This is the Angular headless implementation. See `components/skip-link/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-skip-link></lily-skip-link>
```

## Files

- `SkipLink.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SkipLink.spec.ts` — vitest + TestBed render test
