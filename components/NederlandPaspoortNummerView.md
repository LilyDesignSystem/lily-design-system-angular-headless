# NederlandPaspoortNummerView

a read-only display of Netherlands's Paspoort Nummer

This is the Angular headless implementation. See `components/nederland-paspoort-nummer-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-nederland-paspoort-nummer-view></lily-nederland-paspoort-nummer-view>
```

## Files

- `NederlandPaspoortNummerView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `NederlandPaspoortNummerView.spec.ts` — vitest + TestBed render test
