# EspanaTarjetaSanitariaIndividualInput

an input for entering an España Tarjeta Sanitaria Individual (TSI) unique national healthcare identifier

This is the Angular headless implementation. See `components/espana-tarjeta-sanitaria-individual-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-espana-tarjeta-sanitaria-individual-input></lily-espana-tarjeta-sanitaria-individual-input>
```

- `EspanaTarjetaSanitariaIndividualInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `EspanaTarjetaSanitariaIndividualInput.spec.ts` — vitest + TestBed render test
