# MockupTabletLandscape

a box area that looks like a tablet computer in landscape mode

This is the Angular headless implementation. See `components/mockup-tablet-landscape/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-mockup-tablet-landscape></lily-mockup-tablet-landscape>
```

## Files

- `MockupTabletLandscape.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `MockupTabletLandscape.spec.ts` — vitest + TestBed render test
