# Draft

a draft is a preliminary version of content that is not yet final, often with an in-progress or under-review status.

This is the Angular headless implementation. See `components/draft/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-draft></lily-draft>
```

## Files

- `Draft.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Draft.spec.ts` — vitest + TestBed render test
