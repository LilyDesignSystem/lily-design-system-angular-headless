# ButtonGroup

a wrapper that groups related buttons together

This is the Angular headless implementation. See `components/button-group/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-button-group></lily-button-group>
```

## Files

- `ButtonGroup.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ButtonGroup.spec.ts` — vitest + TestBed render test
