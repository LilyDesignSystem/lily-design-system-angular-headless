# MockupLaptop

a box area that looks like a laptop computer

This is the Angular headless implementation. See `components/mockup-laptop/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-mockup-laptop></lily-mockup-laptop>
```

## Files

- `MockupLaptop.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `MockupLaptop.spec.ts` — vitest + TestBed render test
