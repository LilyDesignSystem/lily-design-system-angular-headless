# GraphicBlock

a wrapper for charts and graphics with title, description, notes, and ARIA description

This is the Angular headless implementation. See `components/graphic-block/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-graphic-block></lily-graphic-block>
```

## Files

- `GraphicBlock.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `GraphicBlock.spec.ts` — vitest + TestBed render test
