# Watermark

a decorative repeating overlay text or image marking a page

This is the Angular headless implementation. See `components/watermark/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-watermark></lily-watermark>
```

## Files

- `Watermark.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Watermark.spec.ts` — vitest + TestBed render test
