# Avatar

an avatar indicator that shows an avatar image or avatar text

This is the Angular headless implementation. See `components/avatar/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-avatar></lily-avatar>
```

## Files

- `Avatar.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Avatar.spec.ts` — vitest + TestBed render test
