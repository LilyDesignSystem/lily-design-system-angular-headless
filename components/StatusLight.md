# StatusLight

a small colored dot status indicator paired with a status label

This is the Angular headless implementation. See `components/status-light/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-status-light></lily-status-light>
```

## Files

- `StatusLight.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `StatusLight.spec.ts` — vitest + TestBed render test
