# PolskaPeselView

a read-only display of Poland's PESEL

This is the Angular headless implementation. See `components/polska-pesel-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-polska-pesel-view></lily-polska-pesel-view>
```

## Files

- `PolskaPeselView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `PolskaPeselView.spec.ts` — vitest + TestBed render test
