# Collapsible

a container that can be expanded or collapsed

This is the Angular headless implementation. See `components/collapsible/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-collapsible></lily-collapsible>
```

## Files

- `Collapsible.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Collapsible.spec.ts` — vitest + TestBed render test
