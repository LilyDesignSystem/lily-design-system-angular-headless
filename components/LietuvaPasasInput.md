# LietuvaPasasInput

an input for entering Lithuania's Pasas

This is the Angular headless implementation. See `components/lietuva-pasas-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-lietuva-pasas-input></lily-lietuva-pasas-input>
```

## Files

- `LietuvaPasasInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `LietuvaPasasInput.spec.ts` — vitest + TestBed render test
