# QrCode

a QR code image generated from text or URL data

This is the Angular headless implementation. See `components/qr-code/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-qr-code></lily-qr-code>
```

## Files

- `QrCode.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `QrCode.spec.ts` — vitest + TestBed render test
