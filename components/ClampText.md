# ClampText

a text container that truncates content to a maximum number of lines

This is the Angular headless implementation. See `components/clamp-text/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-clamp-text></lily-clamp-text>
```

## Files

- `ClampText.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ClampText.spec.ts` — vitest + TestBed render test
