# LiechtensteinPassportNumberView

a read-only display of Liechtenstein's Liechtenstein Passport Number

This is the Angular headless implementation. See `components/liechtenstein-passport-number-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-liechtenstein-passport-number-view></lily-liechtenstein-passport-number-view>
```

- `LiechtensteinPassportNumberView.spec.ts` — vitest + TestBed render test
