# EndNotes

a section of titled endnote items at the end of an article

This is the Angular headless implementation. See `components/end-notes/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-end-notes></lily-end-notes>
```

## Files

- `EndNotes.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `EndNotes.spec.ts` — vitest + TestBed render test
