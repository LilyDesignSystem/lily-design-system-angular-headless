# RomaniaCodNumericPersonalInput

an input for entering Romania's Cod Numeric Personal (CNP)

This is the Angular headless implementation. See `components/romania-cod-numeric-personal-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-romania-cod-numeric-personal-input></lily-romania-cod-numeric-personal-input>
```

- `RomaniaCodNumericPersonalInput.spec.ts` — vitest + TestBed render test
