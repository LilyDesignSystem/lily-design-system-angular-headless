# Card

a grouped content container with header, body, and footer areas

This is the Angular headless implementation. See `components/card/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-card></lily-card>
```

## Files

- `Card.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Card.spec.ts` — vitest + TestBed render test
