# SummaryBox

a boxed callout highlighting key takeaways or next steps from a longer page

This is the Angular headless implementation. See `components/summary-box/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-summary-box></lily-summary-box>
```

## Files

- `SummaryBox.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SummaryBox.spec.ts` — vitest + TestBed render test
