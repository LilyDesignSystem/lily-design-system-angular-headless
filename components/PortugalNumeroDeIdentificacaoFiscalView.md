# PortugalNumeroDeIdentificacaoFiscalView

a read-only display of Portugal's Número de Identificação Fiscal (NIF)

This is the Angular headless implementation. See `components/portugal-numero-de-identificacao-fiscal-view/index.md`
in the canonical repo root for the cross-framework documentation.

