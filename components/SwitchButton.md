# SwitchButton

a toggle switch for turning a setting on or off

This is the Angular headless implementation. See `components/switch-button/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-switch-button></lily-switch-button>
```

## Files

- `SwitchButton.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SwitchButton.spec.ts` — vitest + TestBed render test
