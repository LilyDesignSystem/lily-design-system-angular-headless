# HiddenInput

a hidden input for including data in form submission <input type="hidden">

This is the Angular headless implementation. See `components/hidden-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-hidden-input></lily-hidden-input>
```

## Files

- `HiddenInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `HiddenInput.spec.ts` — vitest + TestBed render test
