# StepList

an ordered list of step items showing progress through a multi-step process

This is the Angular headless implementation. See `components/step-list/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-step-list></lily-step-list>
```

## Files

- `StepList.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `StepList.spec.ts` — vitest + TestBed render test
