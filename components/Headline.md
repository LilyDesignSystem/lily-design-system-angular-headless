# Headline

a page headline with heading, subtitle, and byline area

This is the Angular headless implementation. See `components/headline/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-headline></lily-headline>
```

- `Headline.ts` — standalone Angular 20 component (signal inputs, OnPush)
