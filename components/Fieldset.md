# Fieldset

a group of related form fields with a legend

This is the Angular headless implementation. See `components/fieldset/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-fieldset></lily-fieldset>
```

## Files

- `Fieldset.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Fieldset.spec.ts` — vitest + TestBed render test
