# TileMap

a tile cartogram map with configurable layers for geographic data visualization

This is the Angular headless implementation. See `components/tile-map/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-tile-map></lily-tile-map>
```

## Files

- `TileMap.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `TileMap.spec.ts` — vitest + TestBed render test
