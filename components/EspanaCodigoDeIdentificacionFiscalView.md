# EspanaCodigoDeIdentificacionFiscalView

a read-only display of Spain's Código de Identificación Fiscal (CIF)

This is the Angular headless implementation. See `components/espana-codigo-de-identificacion-fiscal-view/index.md`
in the canonical repo root for the cross-framework documentation.

```html
<lily-espana-codigo-de-identificacion-fiscal-view></lily-espana-codigo-de-identificacion-fiscal-view>
```

- `EspanaCodigoDeIdentificacionFiscalView.spec.ts` — vitest + TestBed render test
