# NorgeFodselsnummerView

a read-only display of Norway's Fødselsnummer

This is the Angular headless implementation. See `components/norge-fodselsnummer-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-norge-fodselsnummer-view></lily-norge-fodselsnummer-view>
```

## Files

- `NorgeFodselsnummerView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `NorgeFodselsnummerView.spec.ts` — vitest + TestBed render test
