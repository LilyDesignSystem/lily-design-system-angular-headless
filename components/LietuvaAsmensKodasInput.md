# LietuvaAsmensKodasInput

an input for entering Lithuania's Asmens kodas

This is the Angular headless implementation. See `components/lietuva-asmens-kodas-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-lietuva-asmens-kodas-input></lily-lietuva-asmens-kodas-input>
```

## Files

- `LietuvaAsmensKodasInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `LietuvaAsmensKodasInput.spec.ts` — vitest + TestBed render test
