# DownloadButton

a download link styled as a button, with optional file size and format metadata

This is the Angular headless implementation. See `components/download-button/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-download-button></lily-download-button>
```

## Files

- `DownloadButton.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `DownloadButton.spec.ts` — vitest + TestBed render test
