# FeatureCard

a large content card with a prominent image positioned alongside or above the text

This is the Angular headless implementation. See `components/feature-card/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-feature-card></lily-feature-card>
```

## Files

- `FeatureCard.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `FeatureCard.spec.ts` — vitest + TestBed render test
