# Toast

a brief auto-dismissing notification message

This is the Angular headless implementation. See `components/toast/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-toast></lily-toast>
```

## Files

- `Toast.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Toast.spec.ts` — vitest + TestBed render test
