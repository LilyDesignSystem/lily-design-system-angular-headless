# SlovenskoRodneCisloInput

an input for entering Slovakia's Rodné číslo (RČ)

This is the Angular headless implementation. See `components/slovensko-rodne-cislo-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-slovensko-rodne-cislo-input></lily-slovensko-rodne-cislo-input>
```

- `SlovenskoRodneCisloInput.spec.ts` — vitest + TestBed render test
