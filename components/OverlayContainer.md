# OverlayContainer

a full-viewport overlay backdrop for modals and sheets

This is the Angular headless implementation. See `components/overlay-container/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-overlay-container></lily-overlay-container>
```

## Files

- `OverlayContainer.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `OverlayContainer.spec.ts` — vitest + TestBed render test
