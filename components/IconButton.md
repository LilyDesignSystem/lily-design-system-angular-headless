# IconButton

a button containing only an icon with a required accessible label

This is the Angular headless implementation. See `components/icon-button/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-icon-button></lily-icon-button>
```

## Files

- `IconButton.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `IconButton.spec.ts` — vitest + TestBed render test
