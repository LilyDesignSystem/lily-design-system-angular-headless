# TabBar

a group of tabs for switching between content panels

This is the Angular headless implementation. See `components/tab-bar/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-tab-bar></lily-tab-bar>
```

## Files

- `TabBar.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `TabBar.spec.ts` — vitest + TestBed render test
