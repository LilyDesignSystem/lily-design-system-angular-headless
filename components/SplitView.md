# SplitView

a two-panel resizable layout container with a draggable divider between them

This is the Angular headless implementation. See `components/split-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-split-view></lily-split-view>
```

## Files

- `SplitView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SplitView.spec.ts` — vitest + TestBed render test
