# Tile

a grouping container that presents related content in a structured format, often as a single clickable area

This is the Angular headless implementation. See `components/tile/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-tile></lily-tile>
```

## Files

- `Tile.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Tile.spec.ts` — vitest + TestBed render test
