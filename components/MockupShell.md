# MockupShell

a box area that looks like a terminal shell

This is the Angular headless implementation. See `components/mockup-shell/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-mockup-shell></lily-mockup-shell>
```

## Files

- `MockupShell.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `MockupShell.spec.ts` — vitest + TestBed render test
