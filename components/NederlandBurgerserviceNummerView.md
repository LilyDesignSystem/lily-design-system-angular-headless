# NederlandBurgerserviceNummerView

a read-only display of Netherlands's Burgerservicenummer (BSN)

This is the Angular headless implementation. See `components/nederland-burgerservice-nummer-view/index.md`
in the canonical repo root for the cross-framework documentation.


- `NederlandBurgerserviceNummerView.spec.ts` — vitest + TestBed render test
