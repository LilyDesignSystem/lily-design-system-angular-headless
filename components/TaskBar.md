# TaskBar

a horizontal bar of task shortcuts or actions

This is the Angular headless implementation. See `components/task-bar/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-task-bar></lily-task-bar>
```

## Files

- `TaskBar.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `TaskBar.spec.ts` — vitest + TestBed render test
