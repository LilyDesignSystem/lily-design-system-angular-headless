# GanttTableTD

a Gantt chart table interactive grid data cell for planning schedule visualization <td>

This is the Angular headless implementation. See `components/gantt-table-td/index.md`
in the canonical repo root for the cross-framework documentation.


- `GanttTableTD.spec.ts` — vitest + TestBed render test
