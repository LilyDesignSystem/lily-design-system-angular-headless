# PhotoPack

a collection of photos displayed together as a group

This is the Angular headless implementation. See `components/photo-pack/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-photo-pack></lily-photo-pack>
```

## Files

- `PhotoPack.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `PhotoPack.spec.ts` — vitest + TestBed render test
