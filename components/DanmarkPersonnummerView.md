# DanmarkPersonnummerView

a read-only display of Denmark's Personnummer (CPR-nummer) (CPR)

This is the Angular headless implementation. See `components/danmark-personnummer-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-danmark-personnummer-view></lily-danmark-personnummer-view>
```

## Files

- `DanmarkPersonnummerView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `DanmarkPersonnummerView.spec.ts` — vitest + TestBed render test
