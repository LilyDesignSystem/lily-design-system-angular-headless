# LiechtensteinNationalIdentityCardNumberInput

an input for entering Liechtenstein's Liechtenstein National Identity Card Number

This is the Angular headless implementation. See `components/liechtenstein-national-identity-card-number-input/index.md`
in the canonical repo root for the cross-framework documentation.
