# SlovenijaEmsoView

a read-only display of Slovenia's Enotna Matična Številka Občana (EMŠO)

This is the Angular headless implementation. See `components/slovenija-emso-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-slovenija-emso-view></lily-slovenija-emso-view>
```

## Files

- `SlovenijaEmsoView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SlovenijaEmsoView.spec.ts` — vitest + TestBed render test
