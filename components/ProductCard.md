# ProductCard

a specialized card for displaying a product with image, title, price, and actions

This is the Angular headless implementation. See `components/product-card/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-product-card></lily-product-card>
```

## Files

- `ProductCard.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ProductCard.spec.ts` — vitest + TestBed render test
