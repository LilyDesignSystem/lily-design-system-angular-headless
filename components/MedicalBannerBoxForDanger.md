# MedicalBannerBoxForDanger

a medical record banner box for danger information e.g. reactions, warnings, alarms, etc.

This is the Angular headless implementation. See `components/medical-banner-box-for-danger/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-medical-banner-box-for-danger></lily-medical-banner-box-for-danger>
```

## Files

- `MedicalBannerBoxForDanger.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `MedicalBannerBoxForDanger.spec.ts` — vitest + TestBed render test
