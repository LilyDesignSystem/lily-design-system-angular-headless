# BelgiqueNumeroDeRegistreNationalView

a read-only display of Belgium's Numéro de Registre National / Rijksregisternummer (NRN)

This is the Angular headless implementation. See `components/belgique-numero-de-registre-national-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-belgique-numero-de-registre-national-view></lily-belgique-numero-de-registre-national-view>
```

## Files

- `BelgiqueNumeroDeRegistreNationalView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `BelgiqueNumeroDeRegistreNationalView.spec.ts` — vitest + TestBed render test
