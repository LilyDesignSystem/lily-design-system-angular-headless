# ContextMenu

a menu that appears on right-click or long-press

This is the Angular headless implementation. See `components/context-menu/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-context-menu></lily-context-menu>
```

## Files

- `ContextMenu.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ContextMenu.spec.ts` — vitest + TestBed render test
