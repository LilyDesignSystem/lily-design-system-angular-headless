# GoToTop

a link that returns users to the top of a long page

This is the Angular headless implementation. See `components/go-to-top/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-go-to-top></lily-go-to-top>
```

## Files

- `GoToTop.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `GoToTop.spec.ts` — vitest + TestBed render test
