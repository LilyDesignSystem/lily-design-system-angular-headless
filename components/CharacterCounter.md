# CharacterCounter

a counter showing remaining or used characters in a text field

This is the Angular headless implementation. See `components/character-counter/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-character-counter></lily-character-counter>
```

## Files

- `CharacterCounter.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `CharacterCounter.spec.ts` — vitest + TestBed render test
