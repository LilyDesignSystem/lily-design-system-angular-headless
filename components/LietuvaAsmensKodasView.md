# LietuvaAsmensKodasView

a read-only display of Lithuania's Asmens kodas

This is the Angular headless implementation. See `components/lietuva-asmens-kodas-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-lietuva-asmens-kodas-view></lily-lietuva-asmens-kodas-view>
```

## Files

- `LietuvaAsmensKodasView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `LietuvaAsmensKodasView.spec.ts` — vitest + TestBed render test
