# FeaturePhoto

a responsive photo with lazy loading and alt text validation

This is the Angular headless implementation. See `components/feature-photo/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-feature-photo></lily-feature-photo>
```

## Files

- `FeaturePhoto.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `FeaturePhoto.spec.ts` — vitest + TestBed render test
