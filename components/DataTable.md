# DataTable

a data table interactive grid for displaying and sorting tabular data <table>

This is the Angular headless implementation. See `components/data-table/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-data-table></lily-data-table>
```

## Files

- `DataTable.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `DataTable.spec.ts` — vitest + TestBed render test
