# KanbanTable

a kanban board table interactive grid for organizing items by status <table>

This is the Angular headless implementation. See `components/kanban-table/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-kanban-table></lily-kanban-table>
```

## Files

- `KanbanTable.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `KanbanTable.spec.ts` — vitest + TestBed render test
