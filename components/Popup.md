# Popup

a temporary overlay that appears above page content

This is the Angular headless implementation. See `components/popup/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-popup></lily-popup>
```

## Files

- `Popup.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Popup.spec.ts` — vitest + TestBed render test
