# SectionHeading

a styled heading introducing a major content section, with optional eyebrow and subtitle

This is the Angular headless implementation. See `components/section-heading/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-section-heading></lily-section-heading>
```

## Files

- `SectionHeading.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SectionHeading.spec.ts` — vitest + TestBed render test
