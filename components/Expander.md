# Expander

a control that expands to reveal more content

This is the Angular headless implementation. See `components/expander/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-expander></lily-expander>
```

## Files

- `Expander.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Expander.spec.ts` — vitest + TestBed render test
