# DateTimeNowInput

an input for entering a date and time and "now" button <input type="date"><input type="time"><button type="button">

This is the Angular headless implementation. See `components/date-time-now-input/index.md`
in the canonical repo root for the cross-framework documentation.

```html
<lily-date-time-now-input></lily-date-time-now-input>
```

- `DateTimeNowInput.spec.ts` — vitest + TestBed render test
