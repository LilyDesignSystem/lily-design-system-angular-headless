# DateTimeLocalInput

an input for entering a date and time without time zone <input type="datetime-local">

This is the Angular headless implementation. See `components/date-time-local-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-date-time-local-input></lily-date-time-local-input>
```

## Files

- `DateTimeLocalInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `DateTimeLocalInput.spec.ts` — vitest + TestBed render test
