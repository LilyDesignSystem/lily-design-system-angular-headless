# EditableForm

a form wrapper for inline editing of content

This is the Angular headless implementation. See `components/editable-form/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-editable-form></lily-editable-form>
```

## Files

- `EditableForm.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `EditableForm.spec.ts` — vitest + TestBed render test
