# HrvatskaOsobniIdentifikacijskiBrojInput

an input for entering Croatia's Osobni identifikacijski broj (OIB)

This is the Angular headless implementation. See `components/hrvatska-osobni-identifikacijski-broj-input/index.md`
in the canonical repo root for the cross-framework documentation.
