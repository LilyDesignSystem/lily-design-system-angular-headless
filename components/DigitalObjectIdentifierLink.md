# DigitalObjectIdentifierLink

a permanent hyperlink for a Digital Object Identifier (DOI) to an electronic source

This is the Angular headless implementation. See `components/digital-object-identifier-link/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-digital-object-identifier-link></lily-digital-object-identifier-link>
```

## Files

- `DigitalObjectIdentifierLink.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `DigitalObjectIdentifierLink.spec.ts` — vitest + TestBed render test
