# RomaniaCodNumericPersonalView

a read-only display of Romania's Cod Numeric Personal (CNP)

This is the Angular headless implementation. See `components/romania-cod-numeric-personal-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-romania-cod-numeric-personal-view></lily-romania-cod-numeric-personal-view>
```

## Files

- `RomaniaCodNumericPersonalView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `RomaniaCodNumericPersonalView.spec.ts` — vitest + TestBed render test
