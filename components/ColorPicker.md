# ColorPicker

a two-dimensional board for picking colors by hue and saturation

This is the Angular headless implementation. See `components/color-picker/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-color-picker></lily-color-picker>
```

## Files

- `ColorPicker.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ColorPicker.spec.ts` — vitest + TestBed render test
