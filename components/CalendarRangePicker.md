# CalendarRangePicker

a picker for selecting a date range on a calendar

This is the Angular headless implementation. See `components/calendar-range-picker/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-calendar-range-picker></lily-calendar-range-picker>
```

## Files

- `CalendarRangePicker.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `CalendarRangePicker.spec.ts` — vitest + TestBed render test
