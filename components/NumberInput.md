# NumberInput

an input for entering a numeric value with validation <input type="number">

This is the Angular headless implementation. See `components/number-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-number-input></lily-number-input>
```

## Files

- `NumberInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `NumberInput.spec.ts` — vitest + TestBed render test
