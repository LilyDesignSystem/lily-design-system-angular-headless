# Icon

a container for displaying an icon

This is the Angular headless implementation. See `components/icon/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-icon></lily-icon>
```

- `Icon.spec.ts` — vitest + TestBed render test
