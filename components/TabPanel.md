# TabPanel

a content panel associated with a tab in a tab bar

This is the Angular headless implementation. See `components/tab-panel/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-tab-panel></lily-tab-panel>
```

## Files

- `TabPanel.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `TabPanel.spec.ts` — vitest + TestBed render test
