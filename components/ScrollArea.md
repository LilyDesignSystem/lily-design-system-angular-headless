# ScrollArea

a scrollable container with custom scrollbar support

This is the Angular headless implementation. See `components/scroll-area/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-scroll-area></lily-scroll-area>
```

## Files

- `ScrollArea.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ScrollArea.spec.ts` — vitest + TestBed render test
