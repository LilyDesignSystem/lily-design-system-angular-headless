# AspectRatioContainer

a container that maintains a fixed aspect ratio

This is the Angular headless implementation. See `components/aspect-ratio-container/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-aspect-ratio-container></lily-aspect-ratio-container>
```

## Files

- `AspectRatioContainer.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `AspectRatioContainer.spec.ts` — vitest + TestBed render test
