# Resizable

a container that the user can resize by dragging

This is the Angular headless implementation. See `components/resizable/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-resizable></lily-resizable>
```

## Files

- `Resizable.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Resizable.spec.ts` — vitest + TestBed render test
