# PolskaNumerIdentyfikacjiPodatkowejInput

an input for entering Poland's Numer Identyfikacji Podatkowej (NIP)

This is the Angular headless implementation. See `components/polska-numer-identyfikacji-podatkowej-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-polska-numer-identyfikacji-podatkowej-input></lily-polska-numer-identyfikacji-podatkowej-input>
```

- `PolskaNumerIdentyfikacjiPodatkowejInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `PolskaNumerIdentyfikacjiPodatkowejInput.spec.ts` — vitest + TestBed render test
