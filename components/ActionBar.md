# ActionBar

a contextual action bar that appears when items are selected, showing the selection count and bulk action buttons

This is the Angular headless implementation. See `components/action-bar/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-action-bar></lily-action-bar>
```

## Files

- `ActionBar.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ActionBar.spec.ts` — vitest + TestBed render test
