# Editable

an inline-editable text element that toggles between view and edit modes

This is the Angular headless implementation. See `components/editable/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-editable></lily-editable>
```

## Files

- `Editable.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Editable.spec.ts` — vitest + TestBed render test
