# Pictogram

an icon-based component pairing an icon with a title and description in a centered or side layout

This is the Angular headless implementation. See `components/pictogram/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-pictogram></lily-pictogram>
```

## Files

- `Pictogram.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Pictogram.spec.ts` — vitest + TestBed render test
