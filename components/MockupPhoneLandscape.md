# MockupPhoneLandscape

a box area that looks like a mobile phone in landscape orientation

This is the Angular headless implementation. See `components/mockup-phone-landscape/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-mockup-phone-landscape></lily-mockup-phone-landscape>
```

## Files

- `MockupPhoneLandscape.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `MockupPhoneLandscape.spec.ts` — vitest + TestBed render test
