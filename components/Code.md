# Code

an inline code span for displaying short code snippets within surrounding text

This is the Angular headless implementation. See `components/code/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-code></lily-code>
```

## Files

- `Code.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Code.spec.ts` — vitest + TestBed render test
