# Splitter

a draggable divider for resizing adjacent panels

This is the Angular headless implementation. See `components/splitter/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-splitter></lily-splitter>
```

## Files

- `Splitter.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Splitter.spec.ts` — vitest + TestBed render test
