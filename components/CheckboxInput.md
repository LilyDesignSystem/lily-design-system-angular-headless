# CheckboxInput

a checkbox input for toggling a boolean value <input type="checkbox">

This is the Angular headless implementation. See `components/checkbox-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-checkbox-input></lily-checkbox-input>
```

## Files

- `CheckboxInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `CheckboxInput.spec.ts` — vitest + TestBed render test
