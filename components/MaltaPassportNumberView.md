# MaltaPassportNumberView

a read-only display of Malta's Malta Passport Number

This is the Angular headless implementation. See `components/malta-passport-number-view/index.md`
in the canonical repo root for the cross-framework documentation.


- `MaltaPassportNumberView.spec.ts` — vitest + TestBed render test
