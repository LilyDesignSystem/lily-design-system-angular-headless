# Button

a generic clickable button element

This is the Angular headless implementation. See `components/button/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-button></lily-button>
```

## Files

- `Button.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Button.spec.ts` — vitest + TestBed render test
