# MedicalBannerBox

a medical banner box that is inside a medical-banner component, using flexbox horizontal, with medical information

This is the Angular headless implementation. See `components/medical-banner-box/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-medical-banner-box></lily-medical-banner-box>
```

## Files

- `MedicalBannerBox.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `MedicalBannerBox.spec.ts` — vitest + TestBed render test
