# NederlandIdentiteitskaartNummerView

a read-only display of Netherlands's Identiteitskaart Nummer

This is the Angular headless implementation. See `components/nederland-identiteitskaart-nummer-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-nederland-identiteitskaart-nummer-view></lily-nederland-identiteitskaart-nummer-view>
```

## Files

- `NederlandIdentiteitskaartNummerView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `NederlandIdentiteitskaartNummerView.spec.ts` — vitest + TestBed render test
