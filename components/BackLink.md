# BackLink

a navigation link to return to a previous page

This is the Angular headless implementation. See `components/back-link/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-back-link></lily-back-link>
```

## Files

- `BackLink.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `BackLink.spec.ts` — vitest + TestBed render test
