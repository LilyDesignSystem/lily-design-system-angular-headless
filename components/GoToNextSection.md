# GoToNextSection

a link that takes users to the next section of the current page

This is the Angular headless implementation. See `components/go-to-next-section/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-go-to-next-section></lily-go-to-next-section>
```

## Files

- `GoToNextSection.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `GoToNextSection.spec.ts` — vitest + TestBed render test
