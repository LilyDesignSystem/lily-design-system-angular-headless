# ActionGroup

a group of action buttons that can collapse to an overflow menu when space is constrained

This is the Angular headless implementation. See `components/action-group/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-action-group></lily-action-group>
```

## Files

- `ActionGroup.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ActionGroup.spec.ts` — vitest + TestBed render test
