# Masonry

a masonry layout container for variable-height items

This is the Angular headless implementation. See `components/masonry/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-masonry></lily-masonry>
```

## Files

- `Masonry.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Masonry.spec.ts` — vitest + TestBed render test
