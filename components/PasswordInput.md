# PasswordInput

an input for entering a password with obscured text <input type="password">

This is the Angular headless implementation. See `components/password-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-password-input></lily-password-input>
```

## Files

- `PasswordInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `PasswordInput.spec.ts` — vitest + TestBed render test
