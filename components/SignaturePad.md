# SignaturePad

a drawing area for capturing a handwritten signature

This is the Angular headless implementation. See `components/signature-pad/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-signature-pad></lily-signature-pad>
```

## Files

- `SignaturePad.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SignaturePad.spec.ts` — vitest + TestBed render test
