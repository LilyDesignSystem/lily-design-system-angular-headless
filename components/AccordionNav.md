# AccordionNav

an accordion navigation area for collapsible accordion information

This is the Angular headless implementation. See `components/accordion-nav/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-accordion-nav></lily-accordion-nav>
```

## Files

- `AccordionNav.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `AccordionNav.spec.ts` — vitest + TestBed render test
