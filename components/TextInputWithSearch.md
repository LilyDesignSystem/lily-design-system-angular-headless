# TextInputWithSearch

a single-line text input field <input type="text"> with search capability

This is the Angular headless implementation. See `components/text-input-with-search/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-text-input-with-search></lily-text-input-with-search>
```

## Files

- `TextInputWithSearch.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `TextInputWithSearch.spec.ts` — vitest + TestBed render test
