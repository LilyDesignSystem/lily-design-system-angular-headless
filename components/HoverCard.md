# HoverCard

a card that appears on hover over a trigger element

This is the Angular headless implementation. See `components/hover-card/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-hover-card></lily-hover-card>
```

## Files

- `HoverCard.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `HoverCard.spec.ts` — vitest + TestBed render test
