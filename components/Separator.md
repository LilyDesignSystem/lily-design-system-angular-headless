# Separator

a horizontal or vertical divider between content sections

This is the Angular headless implementation. See `components/separator/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-separator></lily-separator>
```

## Files

- `Separator.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Separator.spec.ts` — vitest + TestBed render test
