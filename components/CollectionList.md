# CollectionList

a compact list of multiple related items such as articles or events

This is the Angular headless implementation. See `components/collection-list/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-collection-list></lily-collection-list>
```

## Files

- `CollectionList.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `CollectionList.spec.ts` — vitest + TestBed render test
