# PolskaPeselInput

an input for entering Poland's PESEL

This is the Angular headless implementation. See `components/polska-pesel-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-polska-pesel-input></lily-polska-pesel-input>
```

## Files

- `PolskaPeselInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `PolskaPeselInput.spec.ts` — vitest + TestBed render test
