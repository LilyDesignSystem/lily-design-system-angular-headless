# HeroHeadline

a full-bleed hero section with headline overlaid on media

This is the Angular headless implementation. See `components/hero-headline/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-hero-headline></lily-hero-headline>
```

## Files

- `HeroHeadline.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `HeroHeadline.spec.ts` — vitest + TestBed render test
