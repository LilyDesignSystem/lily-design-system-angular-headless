# CurrencyInput

a locale-aware currency input with automatic formatting, symbols, separators

This is the Angular headless implementation. See `components/currency-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-currency-input></lily-currency-input>
```

## Files

- `CurrencyInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `CurrencyInput.spec.ts` — vitest + TestBed render test
