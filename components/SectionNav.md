# SectionNav

a navigation container for section navigation links

This is the Angular headless implementation. See `components/section-nav/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-section-nav></lily-section-nav>
```

## Files

- `SectionNav.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SectionNav.spec.ts` — vitest + TestBed render test
