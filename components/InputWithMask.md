# InputWithMask

an input displaying a format mask placeholder for the user to fill in

This is the Angular headless implementation. See `components/input-with-mask/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-input-with-mask></lily-input-with-mask>
```

## Files

- `InputWithMask.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `InputWithMask.spec.ts` — vitest + TestBed render test
