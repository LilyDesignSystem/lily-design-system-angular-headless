# PinInputDiv

a series of single-digit inputs for entering a PIN or OTP code

This is the Angular headless implementation. See `components/pin-input-div/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-pin-input-div></lily-pin-input-div>
```

## Files

- `PinInputDiv.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `PinInputDiv.spec.ts` — vitest + TestBed render test
