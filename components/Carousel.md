# Carousel

a slideshow for cycling through content items

This is the Angular headless implementation. See `components/carousel/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-carousel></lily-carousel>
```

## Files

- `Carousel.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Carousel.spec.ts` — vitest + TestBed render test
