# InsetText

indented text to distinguish it from surrounding content

This is the Angular headless implementation. See `components/inset-text/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-inset-text></lily-inset-text>
```

## Files

- `InsetText.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `InsetText.spec.ts` — vitest + TestBed render test
