# Slider

a draggable control for selecting a value along a track

This is the Angular headless implementation. See `components/slider/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-slider></lily-slider>
```

## Files

- `Slider.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Slider.spec.ts` — vitest + TestBed render test
