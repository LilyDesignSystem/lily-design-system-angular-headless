# Dial

a rotary dial control for selecting a value

This is the Angular headless implementation. See `components/dial/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-dial></lily-dial>
```

## Files

- `Dial.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Dial.spec.ts` — vitest + TestBed render test
