# SharePage

a component for sharing the page on social media

This is the Angular headless implementation. See `components/share-page/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-share-page></lily-share-page>
```

## Files

- `SharePage.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SharePage.spec.ts` — vitest + TestBed render test
