# Framer

a container for framed content display with decorative border

This is the Angular headless implementation. See `components/framer/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-framer></lily-framer>
```

## Files

- `Framer.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Framer.spec.ts` — vitest + TestBed render test
