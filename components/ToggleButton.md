# ToggleButton

a button that toggles between pressed and unpressed states

This is the Angular headless implementation. See `components/toggle-button/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-toggle-button></lily-toggle-button>
```

## Files

- `ToggleButton.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ToggleButton.spec.ts` — vitest + TestBed render test
