# InfoState

a composition of illustration, title, description, and action for empty, error, or info states

This is the Angular headless implementation. See `components/info-state/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-info-state></lily-info-state>
```

## Files

- `InfoState.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `InfoState.spec.ts` — vitest + TestBed render test
