# FiveStarRatingPickerButton

a picker button for selecting a 1-5 star rating using radio buttons

This is the Angular headless implementation. See `components/five-star-rating-picker-button/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-five-star-rating-picker-button></lily-five-star-rating-picker-button>
```

- `FiveStarRatingPickerButton.spec.ts` — vitest + TestBed render test
