# ResetInput

a button input that resets a form to default values <input type="reset">

This is the Angular headless implementation. See `components/reset-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-reset-input></lily-reset-input>
```

## Files

- `ResetInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ResetInput.spec.ts` — vitest + TestBed render test
