# Listbox

a list of selectable options with keyboard navigation

This is the Angular headless implementation. See `components/listbox/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-listbox></lily-listbox>
```

## Files

- `Listbox.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Listbox.spec.ts` — vitest + TestBed render test
