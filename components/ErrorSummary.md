# ErrorSummary

a summary of all validation errors on a form

This is the Angular headless implementation. See `components/error-summary/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-error-summary></lily-error-summary>
```

## Files

- `ErrorSummary.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ErrorSummary.spec.ts` — vitest + TestBed render test
