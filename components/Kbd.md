# Kbd

show keyboard shortcuts and key combinations in a styled inline element

This is the Angular headless implementation. See `components/kbd/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-kbd></lily-kbd>
```

## Files

- `Kbd.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Kbd.spec.ts` — vitest + TestBed render test
