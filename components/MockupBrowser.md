# MockupBrowser

a box area that looks like a web browser

This is the Angular headless implementation. See `components/mockup-browser/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-mockup-browser></lily-mockup-browser>
```

## Files

- `MockupBrowser.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `MockupBrowser.spec.ts` — vitest + TestBed render test
