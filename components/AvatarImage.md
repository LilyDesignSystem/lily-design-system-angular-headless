# AvatarImage

an avatar indicator inside image such as a user photo

This is the Angular headless implementation. See `components/avatar-image/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-avatar-image></lily-avatar-image>
```

## Files

- `AvatarImage.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `AvatarImage.spec.ts` — vitest + TestBed render test
