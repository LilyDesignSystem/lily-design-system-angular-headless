# MockupWindow

a box area that looks like a desktop window

This is the Angular headless implementation. See `components/mockup-window/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-mockup-window></lily-mockup-window>
```

## Files

- `MockupWindow.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `MockupWindow.spec.ts` — vitest + TestBed render test
