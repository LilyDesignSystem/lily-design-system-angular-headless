# Affix

a wrapper that pins its content to a viewport position while the page scrolls

This is the Angular headless implementation. See `components/affix/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-affix></lily-affix>
```

## Files

- `Affix.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Affix.spec.ts` — vitest + TestBed render test
