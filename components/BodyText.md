# BodyText

a rendered text block within a content width container

This is the Angular headless implementation. See `components/body-text/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-body-text></lily-body-text>
```

## Files

- `BodyText.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `BodyText.spec.ts` — vitest + TestBed render test
