# LietuvaPasasView

a read-only display of Lithuania's Pasas

This is the Angular headless implementation. See `components/lietuva-pasas-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-lietuva-pasas-view></lily-lietuva-pasas-view>
```

## Files

- `LietuvaPasasView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `LietuvaPasasView.spec.ts` — vitest + TestBed render test
