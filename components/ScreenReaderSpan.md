# ScreenReaderSpan

a visually hidden span of text intended for screen readers to provide more descriptive labels

This is the Angular headless implementation. See `components/screen-reader-span/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-screen-reader-span></lily-screen-reader-span>
```

## Files

- `ScreenReaderSpan.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ScreenReaderSpan.spec.ts` — vitest + TestBed render test
