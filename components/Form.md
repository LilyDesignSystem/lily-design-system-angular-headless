# Form

a form element for collecting and submitting user data

This is the Angular headless implementation. See `components/form/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-form></lily-form>
```

## Files

- `Form.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Form.spec.ts` — vitest + TestBed render test
