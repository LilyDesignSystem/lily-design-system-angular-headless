# EmojiCharacterPicker

a picker for browsing and selecting emoji characters

This is the Angular headless implementation. See `components/emoji-character-picker/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-emoji-character-picker></lily-emoji-character-picker>
```

## Files

- `EmojiCharacterPicker.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `EmojiCharacterPicker.spec.ts` — vitest + TestBed render test
