# ContentBlock

a content width constraint container with named column widths

This is the Angular headless implementation. See `components/content-block/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-content-block></lily-content-block>
```

## Files

- `ContentBlock.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ContentBlock.spec.ts` — vitest + TestBed render test
