# Popover

a floating content box anchored to a trigger element

This is the Angular headless implementation. See `components/popover/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-popover></lily-popover>
```

## Files

- `Popover.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Popover.spec.ts` — vitest + TestBed render test
