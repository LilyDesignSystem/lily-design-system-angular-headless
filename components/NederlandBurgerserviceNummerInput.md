# NederlandBurgerserviceNummerInput

an input for entering Netherlands's Burgerservicenummer (BSN)

This is the Angular headless implementation. See `components/nederland-burgerservice-nummer-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-nederland-burgerservice-nummer-input></lily-nederland-burgerservice-nummer-input>
```

## Files

- `NederlandBurgerserviceNummerInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `NederlandBurgerserviceNummerInput.spec.ts` — vitest + TestBed render test
