# EspanaCodigoDeIdentificacionFiscalInput

an input for entering Spain's Código de Identificación Fiscal (CIF)

This is the Angular headless implementation. See `components/espana-codigo-de-identificacion-fiscal-input/index.md`
in the canonical repo root for the cross-framework documentation.
