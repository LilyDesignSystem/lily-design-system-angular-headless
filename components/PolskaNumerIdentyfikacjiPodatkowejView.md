# PolskaNumerIdentyfikacjiPodatkowejView

a read-only display of Poland's Numer Identyfikacji Podatkowej (NIP)

This is the Angular headless implementation. See `components/polska-numer-identyfikacji-podatkowej-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-polska-numer-identyfikacji-podatkowej-view></lily-polska-numer-identyfikacji-podatkowej-view>
```

- `PolskaNumerIdentyfikacjiPodatkowejView.spec.ts` — vitest + TestBed render test
