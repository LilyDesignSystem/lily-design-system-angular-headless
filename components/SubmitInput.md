# SubmitInput

a button input that submits a form <input type="submit">

This is the Angular headless implementation. See `components/submit-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-submit-input></lily-submit-input>
```

## Files

- `SubmitInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SubmitInput.spec.ts` — vitest + TestBed render test
