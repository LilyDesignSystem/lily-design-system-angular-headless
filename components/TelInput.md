# TelInput

an input for entering a telephone number <input type="tel">

This is the Angular headless implementation. See `components/tel-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-tel-input></lily-tel-input>
```

## Files

- `TelInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `TelInput.spec.ts` — vitest + TestBed render test
