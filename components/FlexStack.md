# FlexStack

a flex layout container for vertical or horizontal stacking with consistent gap

This is the Angular headless implementation. See `components/flex-stack/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-flex-stack></lily-flex-stack>
```

## Files

- `FlexStack.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `FlexStack.spec.ts` — vitest + TestBed render test
