# ButtonInput

an input element of type button for form actions <input type="button">

This is the Angular headless implementation. See `components/button-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-button-input></lily-button-input>
```

## Files

- `ButtonInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ButtonInput.spec.ts` — vitest + TestBed render test
