# AvatarText

an avatar indicator inner text such as a user name

This is the Angular headless implementation. See `components/avatar-text/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-avatar-text></lily-avatar-text>
```

## Files

- `AvatarText.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `AvatarText.spec.ts` — vitest + TestBed render test
