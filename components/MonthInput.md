# MonthInput

an input for selecting a month and year <input type="month">

This is the Angular headless implementation. See `components/month-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-month-input></lily-month-input>
```

- `MonthInput.spec.ts` — vitest + TestBed render test
