# MedicalBanner

a prominent message bar across the top of a page, with medical information

This is the Angular headless implementation. See `components/medical-banner/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-medical-banner></lily-medical-banner>
```

## Files

- `MedicalBanner.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `MedicalBanner.spec.ts` — vitest + TestBed render test
