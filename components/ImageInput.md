# ImageInput

a graphical submit button that displays an image <input type="image">

This is the Angular headless implementation. See `components/image-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-image-input></lily-image-input>
```

## Files

- `ImageInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `ImageInput.spec.ts` — vitest + TestBed render test
