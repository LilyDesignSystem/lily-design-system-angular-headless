# SlovenijaEmsoInput

an input for entering Slovenia's Enotna Matična Številka Občana (EMŠO)

This is the Angular headless implementation. See `components/slovenija-emso-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-slovenija-emso-input></lily-slovenija-emso-input>
```

## Files

- `SlovenijaEmsoInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SlovenijaEmsoInput.spec.ts` — vitest + TestBed render test
