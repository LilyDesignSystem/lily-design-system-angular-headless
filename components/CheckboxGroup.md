# CheckboxGroup

a group component that manages a collection of checkboxes with shared state

This is the Angular headless implementation. See `components/checkbox-group/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-checkbox-group></lily-checkbox-group>
```

## Files

- `CheckboxGroup.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `CheckboxGroup.spec.ts` — vitest + TestBed render test
