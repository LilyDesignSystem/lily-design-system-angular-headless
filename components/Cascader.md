# Cascader

a multi-level dropdown for selecting a value from a hierarchy

This is the Angular headless implementation. See `components/cascader/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-cascader></lily-cascader>
```

## Files

- `Cascader.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Cascader.spec.ts` — vitest + TestBed render test
