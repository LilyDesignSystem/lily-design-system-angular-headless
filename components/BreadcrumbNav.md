# BreadcrumbNav

a navigation container for breadcrumb trail links

This is the Angular headless implementation. See `components/breadcrumb-nav/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-breadcrumb-nav></lily-breadcrumb-nav>
```

## Files

- `BreadcrumbNav.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `BreadcrumbNav.spec.ts` — vitest + TestBed render test
