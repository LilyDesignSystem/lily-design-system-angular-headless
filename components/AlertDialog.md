# AlertDialog

a modal dialog for urgent messages requiring user acknowledgment

This is the Angular headless implementation. See `components/alert-dialog/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-alert-dialog></lily-alert-dialog>
```

## Files

- `AlertDialog.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `AlertDialog.spec.ts` — vitest + TestBed render test
