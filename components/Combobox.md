# Combobox

a text input combined with a dropdown list for filtering options

This is the Angular headless implementation. See `components/combobox/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-combobox></lily-combobox>
```

## Files

- `Combobox.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `Combobox.spec.ts` — vitest + TestBed render test
