# DateTimeView

a read-only display of a formatted date and time

This is the Angular headless implementation. See `components/date-time-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-date-time-view></lily-date-time-view>
```

## Files

- `DateTimeView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `DateTimeView.spec.ts` — vitest + TestBed render test
