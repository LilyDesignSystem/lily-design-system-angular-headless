# AddressographBox

a patient identification block printed or rendered from an addressograph-box plate, showing core patient demographics.

This is the Angular headless implementation. See `components/addressograph-box/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-addressograph-box></lily-addressograph-box>
```

## Files

- `AddressographBox.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `AddressographBox.spec.ts` — vitest + TestBed render test
