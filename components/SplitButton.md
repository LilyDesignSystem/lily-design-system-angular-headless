# SplitButton

a button with a primary action and a dropdown for related actions

This is the Angular headless implementation. See `components/split-button/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-split-button></lily-split-button>
```

## Files

- `SplitButton.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `SplitButton.spec.ts` — vitest + TestBed render test
