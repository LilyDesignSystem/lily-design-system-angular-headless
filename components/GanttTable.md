# GanttTable

a Gantt chart table interactive grid for planning schedule visualization <table>

This is the Angular headless implementation. See `components/gantt-table/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-gantt-table></lily-gantt-table>
```

## Files

- `GanttTable.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `GanttTable.spec.ts` — vitest + TestBed render test
