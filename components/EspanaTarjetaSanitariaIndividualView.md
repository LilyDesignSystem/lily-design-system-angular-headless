# EspanaTarjetaSanitariaIndividualView

a read-only display of an España Tarjeta Sanitaria Individual (TSI) unique national healthcare identifier

This is the Angular headless implementation. See `components/espana-tarjeta-sanitaria-individual-view/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-espana-tarjeta-sanitaria-individual-view></lily-espana-tarjeta-sanitaria-individual-view>
```

## Files

- `EspanaTarjetaSanitariaIndividualView.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `EspanaTarjetaSanitariaIndividualView.spec.ts` — vitest + TestBed render test
