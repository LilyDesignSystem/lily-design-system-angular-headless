# RangeInput

a slider input for selecting a value within a range <input type="range">

This is the Angular headless implementation. See `components/range-input/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-range-input></lily-range-input>
```

## Files

- `RangeInput.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `RangeInput.spec.ts` — vitest + TestBed render test
