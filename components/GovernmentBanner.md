# GovernmentBanner

a banner identifying a website as belonging to a government, with an expandable details panel

This is the Angular headless implementation. See `components/government-banner/index.md`
in the canonical repo root for the cross-framework documentation.

## Selector

```html
<lily-government-banner></lily-government-banner>
```

## Files

- `GovernmentBanner.ts` — standalone Angular 20 component (signal inputs, OnPush)
- `GovernmentBanner.spec.ts` — vitest + TestBed render test
